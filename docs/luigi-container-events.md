# Luigi Container Events

This document outlines the events provided by the Luigi Container.<br/>
Some events contain an additional payload which can be utilized for internal objectives.<br/>
In addition you can use standard `addEventListener` function to react on events emmitted by the Luigi Container. Source list of all events can be found [here](https://github.com/luigi-project/luigi/blob/main/container/src/constants/communication.ts).

## Event Reference

### CUSTOM\_MESSAGE&nbsp;  
<p>Event fired when the micro frontend sends a custom message. <br><br>
Payload: <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Object">Object</a></p>

**Properties**

| Name | Type | Description |
| --- | --- | --- |
| data | <code>object</code> | <p>event data</p> |
| id | <code>string</code> | <p>event ID</p> |

  
**Example**  
```js
{
 data: {},
 id: 'some-id'
}
```

* * *

### NAVIGATION\_REQUEST&nbsp;  
<p>Event fired when a navigation has been requested by the micro frontend. <br><br>
Payload: <a href="https://github.com/luigi-project/luigi/blob/main/container/typings/constants/event-payloads.ts">NavigationRequestPayload</a></p>

  
**Example**  
```js
{
 fromClosestContext: false,
 fromContext: null,
 fromParent: true,
 fromVirtualTreeRoot: false,
 link: '/test/route',
 nodeParams: {}
}
```

* * *

### ALERT\_REQUEST&nbsp;  
<p>Event fired when the micro frontend requests to show an alert. <br>
Read more about <code>showAlert</code> params <a href="https://docs.luigi-project.io/docs/luigi-core-api?section=showalert">here</a>. <br><br>
Payload: <a href="https://github.com/luigi-project/luigi/blob/main/container/typings/constants/event-payloads.ts">AlertRequestPayload</a></p>

  
**Example**  
```js
{
 text: 'Custom alert message',
 type: 'info',
 links: {
  goToHome: { text: 'Homepage', url: '/overview' },
  goToOtherProject: { text: 'Other project', url: '/projects/pr2' },
  relativePath: { text: 'Hide side nav', url: 'hideSideNav' },
  neverShowItAgain: { text: 'Never show it again', dismissKey: 'neverShowItAgain' }
 },
 closeAfter: 3000
}
```
