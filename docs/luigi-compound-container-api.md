# Luigi Compound Container API

This document outlines the parameters provided by the Luigi Compound Container. Luigi Compound Container provides the possibility to insert multiple webcomponent-based microfrontends in one container.<br/>
In addition you can use standard `addEventListener` function to react on events emmitted by the Luigi Compound Container. The list of events and their meaning can be found [here](https://docs.luigi-project.io/docs/luigi-container-events).

### new LuigiCompoundContainer()&nbsp;  
<p>Base class for Luigi compound container.</p>

### viewurl&nbsp;  
<p>The URL used for the renderer. <br><br>
Type: <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String">String</a></p>

**Example**  
```js
<luigi-container viewurl="/index.html"></luigi-container>
```
**Example**  
```js
myContainer.viewurl = "/index.html"
```
**Since**: 1.0.0  

* * *

### compoundConfig&nbsp;  
<p>The configuration for the compound microfrontend.
Take a look at the <a href="https://docs.luigi-project.io/docs/navigation-parameters-reference/?section=compound">compound parameter</a> for details. <br><br>
Type: <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Object">Object</a></p>

**Example**  
```js
renderer = { use: 'grid', config: { columns: '1fr 1fr 1fr 2fr', layouts: [{maxWidth: 600, columns: '1fr', gap: 0, ...}]}};
children = [{ viewUrl: '/main.js', context: { label: 'WC', ...}, layoutConfig: {column: '1 / -1', ...}, eventListeners: [{ source: 'input1', ...}}]}];
```
**Example**  
```js
myContainer.compoundConfig = { renderer, children };
```
**Since**: 1.0.0

### webcomponent&nbsp;  
<p>The following properties can be set for the web component object. By default, the web component is set to true. <br><br>
Type: <code>Boolean</code> or <code>WebComponentSettings</code> or <code>String</code></p>


| Param | Type | Description |
| --- | --- | --- |
| [WebComponentSettings] | <code>Object</code> | <p>specifies that the microfrontend is a webcomponent with addtional settings</p> |
| WebComponentSettings.type | <code>String</code> | <p>string, like module</p> |
| WebComponentSettings.selfRegistered | <code>Boolean</code> | <p>if it is true, the web component bundle will be added via script tag</p> |
| WebComponentSettings.tagName | <code>String</code> | <p>tag name where web component is added to DOM</p> |
| string | <code>String</code> | <p>must be a stringified JSON object from type <code>WebComponentSettings</code></p> |

**Example**  
```js
<luigi-container webcomponent="{ type: 'module', selfRegistered: true, tagName: 'my-webcomponent'}"></luigi-container>
```
**Example**  
```js
myContainer.webcomponent = { type: 'module', selfRegistered: true, tagName: 'my-webcomponent'}
```
**Since**: 1.0.0
